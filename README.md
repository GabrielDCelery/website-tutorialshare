# Tutorialshare

## Description

Lightweight personal bookmark manager

## Tools and languages used during the project

	* Front-end
		* Bootstrap
		* AngularJs
	* Back-end
		* Node.js
		* Express.js

## Folder structure

	* config (config files)
	* docs (documentation)
	* middlewares (middlewares for authentication)
	* public
	* routes
	* src
		* log (adding and removing logs)
		* user (authentication)
